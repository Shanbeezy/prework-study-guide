HTML

  • Hypertext Markup Language (HTML), is one of the three cornerstone languages used in web development.
  • The head element contains information (metadata), that helps the browser render the page ccorrectly. Elements included within the head are not visible when a user looks at a webpage.
  • The following element types are commonly used to add content to pages:
    1. Text elements
    2. Image elements
    3. Header elements
    4. List elements
  • Here are some common element types that are used to divide and describe a page:
    1. Meta-related elements
    2. Content division elements
    3. Semantic elements
  • The body element contains the content that is shown to the user. All the elements that we want the user to see and interact with must be placed inside this element.
  • HTML elements work together to build the structure of a webpage and provide a framework that makes it easy for a user to navigate the webpage.

CSS

  • Cascading Style Sheets (CSS) is a programming language used by developers to define how webpage content is presented to users.
  • CSS defines the colors, aesthetics, visual layout, and organization of a webpage
  • There are 3 ways to style a webpage using CSS
    1. Inline CSS - used when a developer wants to style an element directly in the HTML file. These styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only. These are useful for quick and permanent changes but are less flexible, because each inline style must be edited individually whenever you make a design change.
    2. Internal CSS style sheet- Contains CSS rules for the webpage in the head section of the HTML. This makes it possible to embed the entire stylesheet directly in the HTML file. The rules would only apply to that page, and you can configure CSS classes and IDs, which are used to identify elements to style multiple elements in that page's code. All changes to the CSS rule will apply to all the tagged page elements.
    3. External CSS style sheet- This is the most common way to style a webpage because it allows developers to keep all of their CSS rules in a separate file, which makes design changes easier. When using this method, we link the file using a "link" element placed in the HTML's head.
    A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element
  • Do Not Repeat Yourself (DRY) - a coding principle that encourages developers to reuse or share pieces of code in order to reduce the number of lines of code that need to be written or maintained. Can greatly reduce the overall size and complexity of a codebase.

JavaScript

  • JavaScript is a high-level dynamic programming language that allows users to interact with websites. (This includes effects, interactivity, data communication, and more.)
  • A variable is named container that allows us to store data in our code. After declaring a variable, you can give it a value. (Note that variables may hold values that have different data types)
    1. String - This is a sequence of text known as a string. To signify that the value is a string, enclose it in single or double quote markes
    2. Number - This is a number. Numbers don't have quotes around them.
    3. Boolean - This is a True/False value. The words True and False are special keywords that don't need quote marks.
    4. Array - This is a structure that allows you to store multiple values in a single reference.
    5. Object - This can be anything. Everything in JavaScript is an object and can be stored in a variable.
  • An operator is a mathmatical symbol that produces a result based on two values (or variables).
    1. Addition - Add two numbers together or combine two strings.
    2. Subtraction, Multiplication, Division - These do what you'd expect them to do in basic math.
    3. Assignment - This assigns a value to a variable.
    4. Strict equality - This performs a test to see if two values are equal and of the same data type. It returns a True/False (Boolean) result.
    5. Not, Does-not-equal - This returns a logically opposite value of what it precedes. It turns a true into a false, etc.. When it is used alongside the Equality operator, the negation operator rests whether two values are NOT equal.
  • Conditionals are code structures used to test if an expression returns true or not. A very common form of conditionals is the if...else statement.
  • Functions are a way of packaging functionality that you wish to reuse. It's possible to define a body of code as a function that executes when you call the function name in your code. This is a good alternative to repeatedly writing the same code. (If you see something which looks like a variable name, but it's followed by parentheses, it's likely a function.) Functions often take arguments: bits of data they need to do their job. Arguments go inside the parentheses, separated by commas if there is more than one argument.
  • A Forloop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over. Developers must use another statement, such as the conditional if statement to interrpt the control flow by executing a block of code a specific number of times.
  • Real interactivity on a website requires event handlers. These are code structures that listen for activity in the growser, and run code in response. The most obvious example is handling the click event, which is fired by the browser when you click on something with your mouse.
  • Control flow is the order in which a computer executes code in a script. JavaScript is read from top to bottom, and order counts. This means that we must be intentional about how we set up code in a JS file; we can't just throw it all in there any which way!
  • Commenting out code makes it inactive without deleting it, just incase you need it later. To comment out multiple lines of JavaScript code, put (/*) before and (*/) after the block of code.
